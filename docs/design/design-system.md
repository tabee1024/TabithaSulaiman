# Design System Notes

## Purpose

This document defines the visual direction for my portfolio.

My portfolio uses a warm, retro-inspired design system that feels personal, structured, and professional. The design supports clarity, usability, and strong project storytelling.

## Visual Direction

I created the Canva mockups as the starting point for this visual system.

The design is:

- Warm
- Personal
- Retro-inspired
- Playful
- Structured
- Clear
- Professional

The visual style reflects my personality while keeping the site easy to understand and navigate.

## Brand Direction

The design uses:

- Cream background
- Brown interface elements
- Pixel-inspired headings
- Rounded content cards
- Tab-style navigation
- Project carousel or project grid
- Soft contrast between background and content areas
- Clear section separation

The visual system does more than decorate the page. It guides attention, organizes information, and helps visitors move through the portfolio with less friction.

## Typography Direction
Temporary Free Substitution for Hitchcut: Based off Hitchcut, font, I am using Bungee as a free web font alternative because it keeps the headings playful, bold, and personal while staying readable on the web.
The display font for this portfolio is Hitchcut.

Hitchcut creates the retro-inspired visual identity for headings, labels, navigation moments, and selected interface elements.

Long-form content uses a readable sans-serif font. This keeps project explanations, case studies, and technical notes easy to scan.

The typography system uses:

- Bungee for headings and display moments
- A readable sans-serif font for body text
- Larger font sizes for major page sections
- Clear spacing between headings and paragraphs
- Consistent hierarchy across pages

This structure helps visitors quickly distinguish section titles, supporting text, project details, and calls to action.

## Color Direction

The color system uses:

- Cream as the main background color
- Brown as the primary interface color
- Dark text for readability
- Warm neutral card backgrounds instead of pure white
- Soft accent colors for hierarchy and interaction

I chose not to use pure white as the main background because it creates a harsh break from the warm retro-inspired palette. Warm neutrals create a more cohesive interface, reduce visual strain, and maintain strong readability.

Accessibility still remains a priority. Strong contrast is important for users with visual impairments, so the final color system will be tested to make sure text, buttons, links, and interactive elements remain easy to read and identify.

The color system will help vistors:

- Separate sections quickly
- Recognize interactive elements
- Read text comfortably
- Identify important content
- Move through the page without visual confusion

## Layout Direction

The layout uses clear structure and predictable patterns.

The Canva mockups use a card-based layout. Cards group related information, making projects, skills, experience, and case study sections easier to scan.

The layout uses:

- Clear page sections
- Consistent spacing
- Card-based grouping
- Strong alignment
- Predictable navigation
- Responsive behavior for smaller screens

This structure helps visitors understand the site quickly and find relevant information without extra effort.

## Component Direction

The interface includes:

- Header navigation
- Hero section
- About card
- Skill tags
- Project cards
- Project carousel or grid
- Case study sections
- Contact form
- Footer

Each component has a clear purpose. A project card quickly communicates the project name, role, tools, value, and next action.

## UX Principles

The site is designed to be:

- Easy to scan
- Clear on mobile and desktop
- Accessible
- Consistent
- Fast to navigate
- Focused on useful project storytelling

Every design decision supports the user experience. The visual identity gives the portfolio personality while the structure keeps the content clear and easy to use.

## Notes

This design system balances creativity with usability.

Every visual choice supports clarity, storytelling, and trust.
